Adding custom marks

Overview

Checkout this feature in the API documentation

The epeek plug-in allows a basic way of marking genomic locations in the genome browser. This is done using the genome.pin method. Currently, these marks are represented as pins with six different colors. The source code of this theme looks like:

Checkout the complete source code of the theme


var theme = function(gB, div) {
   // We put several pins
   var pin1_url = gB.pin([32900000, 32910000]);
   var pin2_url = gB.pin([32890000]);
   var pin3_url = gB.pin([32880000, 32870000]);

   // We start the genome browser
   gB(div);
   gB.start();
};

In a real example, pin locations may not be hardcoded in the theme's code and come via methods exposed in its API

Everytime the pin method is called, a different pin color is associated with the passed locations. The URLs returned by the methods can be used to write a custom legend for the marks. They can also be used to use the same pin color for new locations. See its API documentation for more details
